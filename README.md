# BMW-Sales-Dashboard

🚗 BMW Sales Analytics Dashboard (Power BI)


📌 Project Overview


This project is an interactive BMW Sales Analytics Dashboard developed using Microsoft Power BI.



The dashboard analyzes global sales data of BMW to provide insights into revenue, sales trends, model performance, and market distribution. It helps in understanding how different factors like region, model type, and fuel type impact overall business performance.



The goal of this project is to convert raw sales data into meaningful visual insights for better decision-making.

🎯 Objectives
Analyze BMW sales performance across different regions

Identify top-performing models and revenue trends

Track Year-over-Year (YoY) and Month-over-Month (MoM) growth

Understand market share and sales distribution

Provide insights for strategic business decisions

🛠️ Tools & Technologies Used
Microsoft Power BI

Power Query (Data Cleaning & Transformation)

DAX (Data Analysis Expressions)

Microsoft Excel

📂 Dataset Description


The dataset includes:

Sales Revenue

Units Sold

BMW Models (3 Series, X Series, etc.)

Region (Asia, Europe, USA, etc.)

Fuel Type (Petrol, Diesel, Electric, Hybrid)

Time Period (Year & Month)

📊 Dashboard Features


🔹 Sales Overview
Total Revenue

Total Units Sold

Average Price per Unit

Overall performance summary



🔹 Model Performance
Sales by different BMW models

Identification of top-performing models

Revenue contribution by each model



🔹 Regional Analysis
Sales distribution across regions

Comparison of global markets

Identification of high-growth regions



🔹 Trend Analysis
Year-over-Year (YoY) growth

Month-over-Month (MoM) trends

Sales performance over time

📈 Key Insights
Certain BMW models generate higher revenue compared to others

Sales are distributed across multiple regions with some leading markets

Increasing trend in electric vehicle (EV) adoption

Stable global performance with balanced market contribution 

🚀 How to Use
Download the .pbix file from this repository

Open it using Power BI Desktop

Use filters and slicers to explore data interactively

📌 Conclusion


This project demonstrates how data analytics can be used to analyze automotive sales performance. It helps in identifying trends, understanding customer preferences, and supporting business strategies using interactive dashboards.
